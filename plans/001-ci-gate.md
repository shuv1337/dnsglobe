## Why this matters

dnsglobe is a published crate (crates.io + Homebrew tap) with a working test
suite (5 tests, all passing), but **nothing runs those tests automatically**.
The only workflow, `.github/workflows/release.yml`, is autogenerated by
cargo-dist and runs `dist plan` on PRs ‚Äî it never executes `cargo test`,
`cargo clippy`, or `cargo fmt`. As a result, a clippy warning has already
shipped on `main`. This plan adds a CI workflow so every later plan
(002‚Äì005) has an enforcement gate, which is why it executes first.

## Steps

### Step 1: Fix the clippy warning in `src/main.rs`

Replace the nested `if` at `src/main.rs:152-157` (excerpt above) by collapsing
the condition into the match guard:

```rust
        KeyCode::Char(c)
            if !modifiers.contains(KeyModifiers::CONTROL)
                && (c.is_ascii_alphanumeric() || matches!(c, '.' | '-' | '_')) =>
        {
            app.insert_char(c.to_ascii_lowercase());
        }
        _ => {}
```

The behavior must be identical: only ASCII alphanumerics and `.`/`-`/`_` are
inserted, lowercased, and only when CONTROL is not held.

**Verify**: `cargo clippy --all-targets -- -D warnings` ‚Üí exit 0.
**Verify**: `cargo test` ‚Üí `5 passed`.

### Step 2: Apply rustfmt

Run `cargo fmt`. Then check the diff:

```
git diff --stat
```

Expected: only `src/dns.rs` and `src/main.rs` change, a handful of lines each
(line-wrapping only ‚Äî no token changes). If rustfmt rewrites `src/resolvers.rs`
(the one-line-per-resolver table), that is unexpected ‚Äî see STOP conditions.

**Verify**: `cargo fmt --check` ‚Üí exit 0, no output.
**Verify**: `cargo test` ‚Üí `5 passed`.

### Step 3: Create `.github/workflows/ci.yml`

Create the file with exactly this content:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:

env:
  CARGO_TERM_COLOR: always

jobs:
  check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: dtolnay/rust-toolchain@stable
        with:
          components: rustfmt, clippy
      - uses: Swatinem/rust-cache@v2
      - name: Format
        run: cargo fmt --check
      - name: Clippy
        run: cargo clippy --all-targets -- -D warnings
      - name: Test
        run: cargo test
```

**Verify**: `python3 -c "import yaml,sys; yaml.safe_load(open('.github/workflows/ci.yml'))"`
‚Üí exit 0 (or, if PyYAML is unavailable, `ruby -ryaml -e "YAML.load_file('.github/workflows/ci.yml')"`;
if neither exists, visually confirm indentation matches the block above exactly).

### Step 4: Full local gate

Run all three commands the workflow will run.

**Verify**:
- `cargo fmt --check` ‚Üí exit 0
- `cargo clippy --all-targets -- -D warnings` ‚Üí exit 0
- `cargo test` ‚Üí `5 passed; 0 failed`
